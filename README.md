# Re:Makeplace-To-Teamcraft

v0.7.2

Download Re:Makeplace here! https://github.com/RemakePlace/App/

Turn your makeplace .json files into an import link usable by teamcraft! This handles all of the nitty gritty details that other tools may miss! This will generate a TC list comprising of every wallpaper, dye, etc, etc. that RMP allows in app but does not index in its own list.

This is a coding mess, I genuinely will take any and all help I learned python specifically for this haha!!

KNOWN ISSUES:
